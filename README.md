# Linkup Utility Hub

享栄高校 OB・OG 会業務などのリスト作業を効率化するための統合ユーティリティツールです。

## 機能
1. **総合照合ツール**: 
   - 「卒業年+氏名」のリストと「確認したい氏名」のリストを照合。
   - 名前の一致を確認し、卒業年を自動転記します。
   - 和暦 (S/H/R) から西暦への自動変換機能付き。
2. **名簿重複確認**:
   - リスト内の重複を検出し、ユニークなリストを作成します。
   - 重複していた名前のリストアップ機能付き。

## 使い方
[Web版を利用する](https://k-hironori.github.io/OB-OG_Utility-Hub/)

1. 上記リンクをクリックしてツールを開きます。
2. 画面上のタブで機能を切り替えて使用してください。
   - 総合照合: 左側のタブ
   - 名簿重複確認: 右側のタブ

## 開発
- HTML/CSS/JavaScript (Vanilla)
- 外部ライブラリ不使用

## ローカルでの実行
`index.html` をブラウザで開くだけで動作します。
